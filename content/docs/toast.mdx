## Installation

Install the Toast component using the nocta-ui CLI:

```bash
npx nocta-ui add toast
```

Then import the component:

```tsx
import { Toast, ToastTitle, ToastDescription, ToastClose } from '@/components/ui/toast';
```

## Multiple Toasts

You can show multiple toasts at the same time. Each toast is independent and can have different content.

<DocsTabs defaultValue="preview">
  <DocsTab value="preview" title="Preview">
    <MultipleToastsDemo />
  </DocsTab>
  <DocsTab value="code" title="Code">
    ```tsx
    const [toasts, setToasts] = useState<{[key: string]: boolean}>({});

    const showToast = (id: string) => {
      setToasts(prev => ({ ...prev, [id]: true }));
    };

    const hideToast = (id: string) => {
      setToasts(prev => ({ ...prev, [id]: false }));
    };

    return (
      <>
        <Button onClick={() => showToast('info')}>Info Toast</Button>
        <Button onClick={() => showToast('success')}>Success Toast</Button>
        
        {toasts.info && (
          <Toast onClose={() => hideToast('info')}>
            <div className="px-6 py-4">
              <ToastTitle>Information</ToastTitle>
              <ToastDescription>This is an informational message.</ToastDescription>
            </div>
            <ToastClose onClose={() => hideToast('info')} />
          </Toast>
        )}
        
        {toasts.success && (
          <Toast onClose={() => hideToast('success')}>
            <div className="px-6 py-4">
              <ToastTitle>Success!</ToastTitle>
              <ToastDescription>Operation completed successfully.</ToastDescription>
            </div>
            <ToastClose onClose={() => hideToast('success')} />
          </Toast>
        )}
      </>
    );
    ```
  </DocsTab>
</DocsTabs>

## Customization

### Custom Styling

The Toast component accepts a `className` prop for custom styling:

```tsx
<Toast 
  className="max-w-md left-4 right-auto"
>
  <div className="px-6 py-4">
    <ToastTitle>Custom Position</ToastTitle>
    <ToastDescription>This toast appears on the left side.</ToastDescription>
  </div>
</Toast>

<Toast className="border-2 border-blue-500">
  <div className="px-6 py-4">
    <ToastTitle>Custom Border</ToastTitle>
    <ToastDescription>This toast has a custom blue border.</ToastDescription>
  </div>
</Toast>
```

### Custom Close Button

Replace the default close button with your own:

```tsx
<Toast onClose={() => setShow(false)}>
  <div className="px-6 py-4">
    <ToastTitle>Custom Close</ToastTitle>
    <ToastDescription>This toast has a custom close action.</ToastDescription>
  </div>
  <button 
    onClick={() => setShow(false)}
    className="absolute top-2 right-2 text-neutral-400 hover:text-neutral-600"
  >
    ✕
  </button>
</Toast>
```

## Accessibility

The Toast component includes proper accessibility features:

- `role="alert"` for screen reader announcements
- Keyboard navigation support
- Focus management
- Semantic HTML structure

```tsx
<Toast 
  onClose={() => setShow(false)}
  aria-label="Success notification"
>
  <div className="px-6 py-4">
    <ToastTitle>Accessible Toast</ToastTitle>
    <ToastDescription>This toast is screen reader friendly.</ToastDescription>
  </div>
  <ToastClose onClose={() => setShow(false)} />
</Toast>
```

## Component Props

### Toast

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | — | Toast content |

| `duration` | `number` | `5000` | Auto-close duration in milliseconds (0 = no auto-close) |
| `visible` | `boolean` | `true` | Controls toast visibility |
| `onClose` | `() => void` | — | Callback when toast closes |
| `className` | `string` | `''` | Additional CSS classes |

### ToastHeader

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | — | Header content |
| `className` | `string` | `''` | Additional CSS classes |

### ToastTitle

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | — | Title content |
| `className` | `string` | `''` | Additional CSS classes |
| `as` | `React.ElementType` | `'h4'` | HTML element to render |

### ToastDescription

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | — | Description content |
| `className` | `string` | `''` | Additional CSS classes |

### ToastActions

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | — | Action buttons |
| `className` | `string` | `''` | Additional CSS classes |

### ToastClose

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `onClose` | `() => void` | — | Close callback function |
| `className` | `string` | `''` | Additional CSS classes |

All components also accept their respective HTML element attributes.
